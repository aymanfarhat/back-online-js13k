## Back online! - JS13k Compo submission

![](game-demo.gif)

[Play it here!](https://back-online-js-13k.firebaseapp.com/)

### Local setup

- Install all npm dependencies: `npm install`
- Launch on localhost: `npm run start`
- Build for production: `npm run build`
